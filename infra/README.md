# Infraestructura

En esta carpeta voy a documentar y versionar la estrategia de Infraestructura como Código del proyecto.

La estrategia principal será usar Terraform con el proveedor de Microsoft Fabric. Si alguna configuración del trial debe hacerse desde la interfaz gráfica, la dejaré documentada con pasos reproducibles, supuestos y evidencias.

## Recursos esperados

- Workspace de Microsoft Fabric.
- Lakehouse con estructura Bronze, Silver y Gold.
- Data Factory Pipeline de Fabric.
- Configuración de alertas o notificaciones.
- Variables separadas del código.
- Evidencia de despliegue o configuración.
