# Generación de datos

En esta carpeta estarán los scripts para generar datos sintéticos del escenario Retail y cargarlos en una base relacional.

Los datos deben ser reproducibles usando una semilla fija y parámetros centralizados en `config/generation_config.yaml`.

## Tablas fuente

- MSTR_ARTICULOS
- MSTR_PROVEEDORES
- MSTR_TIENDAS
- CRM_MIEMBROS
- TRANS_VENTAS
- INV_STOCK_DIARIO
- POST_DEVOLUCIONES
